\chapter{Experiments}
In this chapter I will through experiments investigate what speedup one might get by using the algorithm for parallelizing optimal control problems with  time dependent DE constraints in temporal direction, introduced in previous chapters. Unlike the parallel performance of gradient and objective function evaluation, the parallel performance of our overall algorithm is difficult to model since it depends on the number of gradient and function evaluations that are necessary in the sequential and parallel algorithms.
